# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-26

### Added

- Initial release of the **Vue 3 DDD CQRS Template**.
- **Domain-Driven Design** structure (Entities, Value Objects, Domain Events).
- **CQRS** pattern implementation (Commands, Queries).
- **Clean Architecture** layering (Domain, Application, Infrastructure, UI).
- **Vertical Slice Architecture** organization.
- **Authentication** feature with Supabase.
- **Dashboard** feature placeholder.
- **Logger** service with Console implementation.
- **EventBus** for domain event dispatching.
- Unit Tests setup with Vitest.
- E2E Tests setup with Playwright.
- Comprehensive documentation (`docs/`).
